# TermsLang
The terms programming language

> Simple, Concise, ~~Fast~~

## Latest Version: `v0.8.2`

## Installation
```
cargo install termslang
```

## Run File
```
termslang run example.tms
```

## Format File
```
termslang format example.tms
```

## Debug File
```
termslang debug example.tms
```
