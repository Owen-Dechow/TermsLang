# TermsLang
The terms programming language

> Simple, Concise, ~~Fast~~

## Latest Version: `v0.7.3`

## Installation
```
cargo install termslang
```

## Run File
```
termslang example.tms
```

## Format File
```
termslang example.tms -f
```

```
termslang example.tms --format
```
